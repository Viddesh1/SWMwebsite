Study With Me Website

Study With Me is a simple website that i created which is a To-Do List We can see preview of the SWM(Study With Me) website in PC View and it also have a android application. It is a simple To-Do List web application which has a home page, Sign up page, login page and a Main page which is displayed to the user after successfull authentication is done in login page. In the Main Page there is Stopwatch, Timer, Notification, White Noise and To-Do List.

Front-End used: HTML5, CSS3, JS, Bootstrap, JQuery
Back-End used: Python(Django)
Database used: Sqlite
